# hashnode

Source markdown for blog posts published at **https://prajwalmahajan101.hashnode.dev**.

Each `cl*.md` file is one published post (filename is the Hashnode-generated post ID). Hashnode is the source of truth for what readers see; this repo exists so the drafts and diffs are version-controlled.

## Workflow

- Drafts live here on `main`
- Publish via Hashnode editor or the Hashnode GitHub sync
- Edits and corrections land back here as PRs
